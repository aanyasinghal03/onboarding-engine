🔍 Rule 1 — No Redundant Training
If candidate already has a skill at the required level, no course is recommended. Exact set difference between resume skills and JD requirements.
if resume_level >= jd_level: skip(skill) # No course needed
⬆ Rule 2 — Level Progression
Recommendations always start at the candidate's current level +1. Levels are never skipped: Beginner → Intermediate → Advanced.
start = next_level(current) if known else 'beginner'
🗺 Rule 3 — Topological Sort
Kahn's BFS algorithm resolves prerequisites. Docker always before Kubernetes, Python before Django — dependency-safe ordering guaranteed.
ordered = kahn_bfs(gap, DAG) # O(V+E) · zero violations
🔒 Rule 4 — Catalog Grounding
Every course sourced exclusively from course_catalog.json. Unknown skills use the 'default' real fallback course — never a fabricated name.
courses = CATALOG.filter( skill in c.skills and grounded == True)
